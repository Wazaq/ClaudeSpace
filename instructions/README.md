# Claude App Instructions

This folder contains user-specific instruction files that can be attached to Claude App projects via GitHub.

## Files

- **`myla-clarity.md`** - Myla's Clarity financial system usage guide
- **`brent-clarity.md`** - Brent's Clarity financial system reference

## How to Use

1. In Claude App, create/edit your project
2. Attach the relevant instruction file from this GitHub repo
3. Claude App will load these instructions alongside your custom instructions

## Benefits

- ✅ Single source of truth (update once, works everywhere)
- ✅ Git version controlled (track changes, sync across devices)
- ✅ No duplication between CLI and App
- ✅ Easy to update as features are added

## Updating Instructions

1. Edit the relevant `.md` file in this folder
2. Commit and push to GitHub
3. Claude App will use the updated version next session
